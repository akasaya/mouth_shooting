# MOUTH SHOOTING

マウスだけで遊ぶブラウザ・シューティング。
依存ゼロ・ビルド不要の素の HTML/Canvas/JavaScript で作られており、ブラウザで開くだけで遊べます。

## 遊び方

| 操作 | 内容 |
|------|------|
| マウス移動 | 自機がカーソルに追従して動く |
| 左クリック（長押しで連射） | **ショット**。最寄りの敵へオートエイム。コンボを積む準備役 |
| 右クリック（長押しで溜め→離す） | **ボム**。自機中心から衝撃波の輪が拡大。半径は溜め時間に比例。溜め中はショット不可・移動が遅くなる |

### コツ（ゲームの肝）

- ショットの稼ぎは小さい。**コンボを積むほどボムが速く溜まり**、音楽のレイヤーが増えて盛り上がります。
- 大きく稼ぐのは**ボムの範囲一掃**。同時撃破でコンボが跳ね上がります。
- ただしステージが進むと敵数・突進速度・弾数が増え、**ボム頼みだと溜め中の隙が致命傷**に。
  ショット（コンボ維持）とボム（大量得点）を編むのが攻略の軸です。
- 被弾するとコンボは消えます。ライフ 0 でゲームオーバー。ハイスコアはブラウザに保存されます。

## ローカルで動かす

ES Modules を使うため、`file://` で直接開くのではなくローカルサーバ経由で開きます。

```bash
python3 -m http.server 8000
# ブラウザで http://localhost:8000 を開く
```

初回はタイトル画面をクリックして開始してください（クリックで音が有効になります）。

## 開発

- **テスト**（純粋ロジック: コンボ計算・当たり判定・ボム半径など）:
  ```bash
  node --test
  ```
- 当たり判定・描画・音・入力は関心ごとに分離しています（`src/` 参照）。
- 手触り・難易度の数値は `src/config.js` に集約しています。

### 構成

```
index.html / style.css     画面とエントリ
src/
  main.js     ゲームループ・状態機械・結線
  config.js   調整用の数値（速度・チャージ・難易度カーブ）
  input.js    マウス入力
  player.js   自機（追従・連射）
  bullet.js   自機弾・敵弾
  enemy.js    敵・突進・射撃・ボス・被弾処理
  bomb.js     チャージ・衝撃波
  combo.js    コンボ計数と派生報酬（純粋関数）
  collision.js 当たり判定（純粋関数）
  stage.js    ステージ・難易度カーブ
  audio.js    効果音 + コンボ連動の動的音楽（WebAudio 生成）
  particles.js / draw.js / hud.js  描画
tests/        node:test による純粋ロジックのテスト
```

## 公開

GitHub Pages（静的配信）で公開できます。リポジトリ Settings → Pages で
ブランチ `main` / フォルダ `/ (root)` を選ぶだけで、`https://<user>.github.io/mouth_shooting/` が発行されます。
